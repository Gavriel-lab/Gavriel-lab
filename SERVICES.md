## Repo Onboarding Audit

Price range: USD 80-120

Deliverables:

- A concise audit of the README, install steps, development setup, and first-run path
- Exact broken commands, missing prerequisites, stale screenshots, or confusing steps
- A prioritized fix list
- Optional PR for small documentation fixes when the scope is clear

Best for:

- Projects with active users but confusing setup docs
- AI tooling repos where new contributors need a clean first-run path

## README / Quickstart Cleanup

Price range: USD 80-150

Deliverables:

- Cleaned README structure
- Verified install, build, test, and demo commands
- Short troubleshooting section for common setup failures
- Minimal wording changes that preserve the maintainer's voice

Best for:

- Tools with a useful product but outdated docs
- Repos receiving repeated "how do I run this?" issues

## Frontend UI Triage

Price range: USD 120-200

Deliverables:

- A focused pass on one screen or flow
- Layout, spacing, responsive behavior, and accessibility notes
- Minimal PR if the requested changes are small and verifiable
- Screenshots or before/after notes

Best for:

- React/TypeScript dashboards, catalogs, tools, and admin screens
- Projects that need polish without a full redesign

## Storybook / Component Example

Price range: USD 150-250

Deliverables:

- One or two useful stories for an existing component
- Common states, edge cases, and interaction notes
- Small cleanup only when needed for the stories to run

Best for:

- Design-system repos
- Apps with reusable UI components but weak review previews

## Notes

- I do not take security-sensitive, payment-sensitive, or private-data tasks without a clear permission boundary.
- I do not spam maintainers with generic pitches.
- I prefer a short public issue or bounty thread for scope alignment before work starts.
